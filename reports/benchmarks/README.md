# Benchmark Reports

Benchmark outputs generated by `src/real_trace_sa_benchmark.py` live here.

Expected files:

- `real_trace_sa_benchmark.json`
- `real_trace_sa_benchmark.csv`
- `summary.json`
- `summary.csv`
- `summary.md`

The benchmark script may also emit derived comparison fields for brute force and CP-SAT when the instance is small enough to support them.

Benchmark windows live under `data/windows/benchmarks/` and are generated by `src/benchmark_window_generator.py`.
