# Trace Windows

Generated inputs for experiments live here.

- `sample_trace_window.json`: overlap window generated from the real trace
- `sample_trace_window_start_only.json`: start-time-filtered window used for mapping-only experiments
- `sample_trace_window_90m_start_only.json`: smaller start-only window for tighter validation
- `sample_trace_window.py`: Python module form of the overlap window for import-based experiments

These files are generated by `src/real_trace_window_generator.py`.
