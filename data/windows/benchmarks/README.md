# Benchmark Windows

This directory holds trace windows prepared for the phase 5.5 benchmark sweep.

Generated by `src/benchmark_window_generator.py`.

Naming pattern:

- `cpu-only_10.json`
- `cpu-only_20.json`
- `cpu-only_30.json`
- `gpu-only_10.json`
- `gpu-only_20.json`
- `gpu-only_30.json`
- `mixed_10.json`
- `mixed_20.json`
- `mixed_30.json`

Each file uses the shared cluster inventory:

- 410 CPU nodes
- 12 GPU nodes
- 128 CPU capacity per node
- 4 GPU capacity on GPU nodes
